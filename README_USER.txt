【系统要求】

- Windows 7 或更高版本
- 64位操作系统

═══════════════════════════════════════════════════════════

【重要说明】

✅ 本游戏是独立程序，无需安装 Python 或任何依赖库
✅ 解压后即可直接运行
✅ 所有依赖已打包在 _internal 目录中
⚠️ 请勿删除 _internal 目录，否则程序无法运行

═══════════════════════════════════════════════════════════

【游戏说明】

这是一款电路板修复益智游戏。

玩法：
- 点击黑色瓦片旋转电路元件
- 连接电源到终端即可过关
- 使用最少的移动次数获得更高评分

控制：
- 鼠标左键：点击瓦片旋转
- F3 或 D 键：切换调试信息
- ESC 键：暂停游戏

═══════════════════════════════════════════════════════════

【卸载】

直接删除整个游戏目录即可。
游戏不会在系统中留下任何文件。
